# PzomPatch
이 프로그램은 프로젝트 좀보이드 간편 패치 윈도우 데스크톱 어플리케이션입니다.
</p>


## 기능
- 클라이언트 메모리 사이즈 조정 ( ProjectZomboid64.bat , ProjectZomboid64.json )
- **좀비 증발 Fix**(No Mo Culling)와 **더나은FPS**(BetterFPS)모드 패치
- ServerListSteam 서버 추가
